HTML

  • HTML is Hypertext Markup Language for short, it is used to create the structure of a webpage, provide content and basic framework.
  • An element is a type of content on a webpage.
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.

Git

  • Git is a command line interface, CLI, which is a program that allows us to input text commands to execute operating system functions.
  • These commands may look like "git status" which checks the branch we are currently using.
  • "git checkout -b branch-name" creates a new branch and switches to it.
